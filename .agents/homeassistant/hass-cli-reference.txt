=== COMMAND: state ===
Usage: hass-cli state [OPTIONS] COMMAND [ARGS]...

  Get info on entity state from Home Assistant.

Options:
  --help  Show this message and exit.

Commands:
  delete    Delete entity state from Home Assistant.
  edit      Edit entity state from Home Assistant.
  get       Get/read entity state from Home Assistant.
  history   Get state history from Home Assistant, all or per entity.
  list      List all state from Home Assistant.
  toggle    Toggle state for one or more entities in Home Assistant.
  turn_off  Turn entity off.
  turn_on   Turn entity on.

--- SUBCOMMAND: state history ---
Usage: hass-cli state history [OPTIONS] ENTITIES...

  Get state history from Home Assistant, all or per entity.

  You can use `--since` and `--end` to narrow or expand the time period.

  Both options accepts a full timestamp i.e. `2016-02-06T22:15:00+00:00` or a
  relative expression i.e. `3m` for three minutes, `5d` for 5 days. Even `3
  minutes` or `5 days` will work. See
  https://dateparser.readthedocs.io/en/latest/#features for examples.

Options:
  --since TEXT  Start of the period to get history from. A timestamp or
                relative expression relative to now. Defaults to 1 day.
  --end TEXT    End of the period to query history from. A timestamp or
                relative expression relative to now. Defaults to now.
  --help        Show this message and exit.
